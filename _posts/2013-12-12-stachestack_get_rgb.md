The `stachestack_get_rgb` function has been deprecated since StacheStack 3.2

It has now moved inside the [`StacheStack_Color`](/classes/StacheStack_Color) class.

```php
<?php return StacheStack_Color::get_rgb( $hex, $implode ); ?>
```
